## SLIDE 1: Hook / Opening (ENHANCE)

### Current State

### Changes Required

**Position:** Top 20% of slide, centered horizontally

```

```


**Rationale:** Placeholder untuk owner yang ingin langsung lihat. Tidak mengganggu visual utama.

## Visual Consistency Notes

### Demo Link Styling
```
Position: Bottom right corner
Margin: 16px from right edge, 16px from bottom edge
Font: Caption size (10-12pt)
Color: Muted, sesuai warna sekunder deck
Format: "demo: app-citradesa.vercel.app"
```

### "So What" Box Styling
```
Background: Subtle fill sesuai warna deck atau transparent
Border: 1px subtle border (optional)
Padding: 16px internal
Header: "UNTUK OWNER:" in bold/caps
Body: Regular text, 2-3 lines max
```

### New Slide Template
```
Use existing slide template
Maintain color scheme consistency
Headlines: Same font/size as existing slides
Body text: Same font/size as existing slides
Layout: Same grid/spacing as existing slides
NO ICONS - text only per deck visual style
```

### Typography
```
Headlines: Use exact same font family/weight/size as existing slides
Body: Use exact same font family/weight/size as existing slides
Callout headers (UNTUK OWNER): Bold/caps, same font family
All text: Follow existing deck typography exactly
```
